Frequently Asked Questions

Are All the mods are available on GPL MODS are safe to use/download +

Safty is our first priority to our every user we always check our mods with multiple antivirus and all mods are provided here are more then 14-days tested by our community members.

How do I install/downloard (.XAPK file)? +

.XAPK file can be downloaded and installed by using GPL Mart. As .XAPK file is a special type of android file which contains apk + obb and can't be installed by using the android package installer.

How do I install (.APK file)? +

To install an APK file, first you need to download an apk file, if you are using chrome or any other browsers then go to download section or where you store your apk file, click on the apk file an popup will appear asking you to enable unknown sources. Click on settings and enable the unknown sources and click on install button to install apk file.

A mod I downloaded isn't working. What should I do? +

First, Check the mod you have downloaded is made for your device, if you have downloaded ios mod it will not work for android devices and vice-versa, if you have downloaded jailbroken ios mods it will not work on jailed devices, For wordpress mods please check that the plugin/theme is compatible with your wordpress version and same for wimdows.

How to install WordPress (.zip) file. +

First, you need to login to your wordpress dashboard then go to plugins then add new plugin then select upload .zip and select your zip file, if the plugin is a package, a package contains 2 or more plugins first you need to extract it by using software like winrar,7-zip then you install it one by one.

How do I install a (.deb) file? +

The deb tweaks can be installed on jailbroken apple devices only and they are not for debian linux. If your device is jailbroken then you can use them and install it by using your package installer like cydia,zebra,sileo etc and XinamineAIO can be used to convert rootfull tweeks to rootless and vise-versa.

How do I install an (.ipa) file. +

You can directly install .ipa files on your jailed/jailbroken apple devices directly by using GPL Mart or you can use tools like sidestore, ksign, scarlet, trool stote etc.

How do I upload a mod to GPL? +

You can share your mods after registration to our website after registration you can upload your mods by going to your profile and then mod upload section or you can go to our homepage to upload your mods your mod will be tested and if it passes all the test then it will be available for download.
