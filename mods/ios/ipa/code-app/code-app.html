About This App

Code App is a desktop-class code editor for iPadOS with built-in runtimes for Node.js, Python, C, C++, PHP, and Java. It provides a robust development environment right on your iPad.

Key Features

  • A robust, high-performance text editor (Monaco Editor from VS Code).
  • Autocomplete and static code analysis for Python & Java.
  • First-class local file system support.
  • SSH / FTP remote connection support (Files and terminal).
  • Embedded emulated terminal and Git Version Control.
  • Package manager support (pip and npm).
