About This Plugin

WP Dark Mode instantly creates a stunning dark mode version of your website without any complicated setup. It's the easiest and highest-rated dark mode plugin for WordPress.

Core Features

  • Floating Dark Mode Switch: Display a dark mode toggle in the footer of your website.
  • OS-Based Dark Mode: Automatically detects the user's device theme and enables dark mode.
  • Admin Panel Dark Mode: Give your eyes a rest while working in the WordPress dashboard.
  • Shortcode Support: Display the dark mode switch anywhere using a simple shortcode.

Go Beyond with WP Dark Mode Ultimate

  • Schedule Dark Mode: Set a time for dark mode to automatically enable on your website.
  • Premium Color Presets: Access 13+ exclusive ready-made dark mode color presets.
  • Dark Mode-Based Image/Video Replacement: Replace light-mode media with different files in dark mode.
  • Exclude Pages & Elements: Prevent specific pages or CSS elements from changing into dark mode.
  • Dark Mode Analytics: Track how many visitors are using the dark mode on your site.
